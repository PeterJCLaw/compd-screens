The competition daemon screen manager.

Getting Started
===============

To install:

1. Install compd
2. Run *./install*

All other dependencies, through the magic of virtualenv, are installed locally - in the *dep* directory as a matter of fact.

To run, use *./run*.

To develop, enter the virtualenv using *./shell*.
